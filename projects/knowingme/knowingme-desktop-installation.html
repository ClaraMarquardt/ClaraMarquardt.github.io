KnowingMe

You Are What You Write

Follow the below instructions for Mac or Windows
to install and launch the KnowingMe desktop app.

  macOS  ||  Windows























macOS Setup

1) Install Docker

  • Download Docker
  • Follow the Docker installation instructions

2) Install Kitematic

  • Download and install Kitematic

3) Launch the KnowingMe Desktop App

  • Launch Kitematic.
  • Login using the following credentials: username = knowingmeapp / password = knowingme2018
  • Navigate to "My Repos" (top right-hand corner)
  • Click "Create" on the 'knowingmeaws' repository
  • Click on the folder listed under "Volumes" (bottom right-hand corner). If a pop-up appears select "Enable Volumes"
  • Navigate to "Settings" (top right-hand corner), then navigate to the "Volumes" tab. Press "Change". Select or create a directory in which to store any user data
  • Return to the home screen ("Home" in the top right-hand corner). Click the icon on the right-hand side of "Web Preview" (top right-hand corner) to launch the app
  • The app should open in your browser (you may need to reload the website after a few seconds). When a pop-up window appears, log-in using the following credentials: username = knowingmeapp / password = knowingme2018

4) Terminate the KnowingMe Desktop App

  • Terminate the app by closing the browser window and pressing “Stop” in Kitematic (top left-hand corner). Delete all user data by trashing the folder which you designated during the set-up process

*) Notes & Troubleshooting

  • If a problem arises when launching or relaunching Kitematic: Ensure that the option "Use VirtualBox instead of Native on next restart" is unchecked in the 'Settings' tab (accessed by clicking the wheel icon in the lower left-hand corner of Kitematic)
  • If a "permission denied" error occurs soon after launching the app: Ensure that the folder which you selected to store any user data is contained within your home directory ("/User/[username]/") and that there are no spaces in the path to the folder
